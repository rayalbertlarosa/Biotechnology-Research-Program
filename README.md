# Biotechnology Research Dashboard

A professional web-based dashboard for managing and presenting research projects at De La Salle Medical and Health Sciences Institute (DLSMHSI).

## Features

- **Four Research Projects**: Project Harper, Project Bioregrade, Project Taal Volcano, and Project Cancer Research
- **Interactive Dashboard**: View detailed project information, timelines, and milestones
- **Professional Design**: La Sallian green color scheme with modern UI/UX
- **Project Logo Placeholders**: Visual branding for each research project
- **Contact Information**: Direct links to program leader contact details
- **Responsive Layout**: Works on desktop and mobile devices

## Live Demo

Simply open `index.html` in a web browser to view the dashboard.

## Local Development

### Using Python (Recommended)

```bash
# Navigate to project directory
cd "Research-Project Presentation Software"

# Start local server
python -m http.server 8000

# Open browser to http://localhost:8000
```

### Direct File Access

Alternatively, you can directly open `index.html` in your web browser.
